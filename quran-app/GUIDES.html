🚀 Memulai

Selamat datang di Al-Quran Digital! Aplikasi web modern untuk membaca dan mendengarkan Al-Quran dengan mudah dan nyaman.

Langkah Pertama:

1. Buka Aplikasi Buka file index.html di browser Anda atau akses melalui server lokal.
2. Pilih Surah Klik pada surah yang ingin dibaca di sidebar sebelah kiri. Surah akan dimuat bersama dengan teks Arab, transliterasi, dan terjemahan Indonesia.
3. Baca atau Dengarkan Baca ayat-ayat atau tekan tombol "Putar" untuk mendengarkan audio dari pembaca pilihan Anda.

💡 Tips & Trik

Menggunakan Pembaca Berbeda

Anda dapat memilih dari 4 pembaca Al-Quran yang berbeda:

  • Abdul Basit
  • Mishari Al-Afasy
  • Maher Al-Muaiqly
  • Muhammad Siddiq Al-Minshawi

Pilih pembaca favorit dari dropdown di audio player.

Menyalin Ayat

Klik tombol "Salin" pada ayat untuk menyalin teks Arab ke clipboard. Anda kemudian dapat mempaste ke aplikasi atau dokumen lain.

Berbagi Ayat

Gunakan tombol "Bagikan" untuk membagikan ayat ke media sosial atau aplikasi messaging. Jika tidak didukung, link akan disalin ke clipboard.

Mode Gelap

Klik ikon bulan di sudut kanan atas untuk mengaktifkan/menonaktifkan dark mode. Preferensi Anda akan disimpan secara otomatis.

🛠️ Troubleshooting

Audio Tidak Terputar

Solusi:
  • Periksa koneksi internet Anda
  • Coba ubah pembaca Al-Quran dari dropdown
  • Refresh halaman (Ctrl+R atau Cmd+R)
  • Coba browser berbeda

Halaman Lambat

Solusi:
  • Periksa kecepatan koneksi internet
  • Gunakan browser yang lebih update
  • Bersihkan cache browser (Ctrl+Shift+Delete)
  • Tutup tab atau aplikasi lain yang menggunakan bandwidth

Dark Mode Tidak Tersimpan

Solusi:
  • Pastikan Local Storage diaktifkan di browser
  • Cek pengaturan privasi/incognito mode
  • Refresh halaman dan coba lagi

🔧 Teknologi yang Digunakan

Frontend: HTML5, CSS3, JavaScript ES6+
API: Al-Quran Cloud API (https://api.alquran.cloud)
Storage: Local Storage untuk preferensi user
Offline: Service Worker untuk support offline

📞 Bantuan & Dukungan

Jika mengalami masalah atau memiliki pertanyaan:

  1. Buka Console browser (F12) untuk melihat pesan error
  2. Cek koneksi internet Anda
  3. Coba hard refresh (Ctrl+Shift+R atau Cmd+Shift+R)
  4. Bersihkan cache dan cookies browser
  5. Gunakan browser berbeda untuk testing
